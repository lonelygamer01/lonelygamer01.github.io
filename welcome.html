How to play the game

Before you could play the game You must enter a username that way the game can store you records so the data won't get lost after closing the browser.
Please keep in mind that the data is stored locally so switching browsers won't transfer the data!

After entering into the game You can see a timer which obviously measuring the time you spend finishing the game.
The button with a little flag icon is an indicator whether the flag is enabled or not. It only works when the game started.
The restart button cancels the current session and reset the timer too.

You can choose from 3 difficulties:
Easy (8 X 8) 10 mines
Medium (14 X 14) 30 mines
Hard (20 X 20) 40 mines
Once you choose the difficulty the game starts instantly!

The records button shows You the last game a player won, and the best game since the last delete.
If you press the button again the tab will close.

Have fun and wish You much luck playing with this game!
